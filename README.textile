
h2. PyLucid 'Weave' glue Plugin

A tiny PyLucid glue plugin to use django-weave in PyLucid as a PluginPage.

Complete install instuctions here:

http://www.pylucid.org/permalink/324/howto-support-the-django-weave-app

h2. info

django-weave is a reusable application which implements a Mozilla Labs Weave server for Django.
* Homepage: http://code.google.com/p/django-weave/ 
* Sourcecode: http://github.com/jedie/django-weave

h2. setup

Quick how to install the plugin into pylucid:

pre. ~$ mkdir pylucid_plugins
~$ cd pylucid_plugins
~/pylucid_plugins$ git clone git://github.com/jedie/PyLucid-Weave-plugin.git
~/pylucid_plugins$ ln -s ~/pylucid_plugins/PyLucid-Weave-plugin/pylucid_weave ~/PyLucid_env/src/pylucid/pylucid_project/external_plugins/

If you don't have git, but suversion, you can checkout with this command:

pre. ~/pylucid_plugins$ svn checkout http://svn.github.com/jedie/PyLucid-Weave-plugin.git

how to update it:

pre. ~$ cd pylucid_plugins/PyLucid-Weave-plugin
~/pylucid_plugins/PyLucid-Weave-plugin$ chmod +x update.sh
~/pylucid_plugins/PyLucid-Weave-plugin$ ./update.sh

the update script runs "git pull origin master" or "svn update"
